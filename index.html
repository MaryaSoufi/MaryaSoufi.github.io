Sarnex — Investment Dashboard Redesign

Role: UX/UI & Product Designer

Tools: Figma, Cursor, GitHub

AREX Capital Investment UX Dashboard Figma

Overview: Redesigned a web-based investment dashboard to simplify portfolio management and make complex financial data easy to interpret. The preview is a redacted layout study, inspired by the live platform.

The challenge: Investment dashboards often overwhelm users with data. Clients struggled to quickly access portfolio insights, understand performance trends, and act on key metrics.

Open the case study for approach and impact.

Case study

My approach

  • Mapped user journeys and pain points to streamline navigation and reduce friction.
  • Designed dashboards with clear hierarchy and visual emphasis on critical portfolio metrics.
  • Built interactive components and collaborated with developers to ensure usability.
  • Iterated designs based on user and stakeholder feedback, following UX best practices.

Impact

  • Users can now access insights more efficiently, reducing cognitive load.
  • Financial data is easier to interpret thanks to visual hierarchy and intuitive layout.
  • Contributed to a scalable, user-centered dashboard framework for future updates.
